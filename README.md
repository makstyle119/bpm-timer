# BPM Timer

A small static BPM timer/metronome built with HTML, CSS, and JavaScript.

## Features

- Set BPM from 30 to 240
- Run for a fixed number of minutes
- Run infinitely
- Tap BPM detection
- Browser-only audio using the Web Audio API
- No backend and no build step

## Use

Open `index.html` in your browser, or deploy the repo with GitHub Pages.
